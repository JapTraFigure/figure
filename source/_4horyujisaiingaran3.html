Now, we will add electric lights.

This diorama is very large, so it is very difficult to install the LEDs one by one in the temple.It is very difficult to install LEDs one by one. I could have used tape LEDs as usual, but it would have been quite difficult because of the many curved lines. So, I decided to use illumination lights this time. This time, we decided to use illumination lights, which are used to wrap around Christmas trees and so on.

This is easy to place because it can be bent freely. Various types of LEDs are sold on Amazon.com, but most of them have 10cm LED spacing. LED spacing is 10cm. In dioramas, we use the type with 5cm LED spacing because it extends the length of the diorama. The following product is 1.2m long, has 24 LEDs, and comes with button batteries. It contains 10 of these and is extremely inexpensive.We recommend this product because it can be used for a variety of decorations and is very useful.



But I wonder if they can make a profit with 240 LEDs and 20 3V button batteries in the box. I wonder if there is any profit in this price.

The lead wires can be fixed quickly by gluing them together with a glue gun.

A glue gun and stick cost 300 yen (purchased at a 100 yen store). The glue gun alone used to cost about 800 yen. So, it is really a good time!

Next, sand was sprinkled on the ground and thinned wood bond was sprayed to fix it in place.

After sprinkling sand, the wiring and LEDs are no longer noticeable!
